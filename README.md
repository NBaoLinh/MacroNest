## Features

*   ⌨️ **Macros**: Central engine to automate your workflow. **All features listed below can be easily triggered and orchestrated through macros.** Supports key sequences, mouse actions, and loops.
*   ➕ **Crosshair**: Customizable visual center overlay.
*   🪟 **Window Control**: 
    *   Presets: Force specific window sizes and positions.
    *   Focus: Automatically switch focus to target windows.
*   📜 **Commands**: Execute PowerShell or CMD scripts directly.
*   🏷️ **HUD (Labels)**: Create and display custom text overlays.
*   👁️ **Vision**: 
    *   Image Search: Detects visual templates on screen.
    *   Pixel/Color: Monitors specific pixels or regions for color changes.
*   📌 **Pin**: Toggle "Always on Top" for any window.
*   🖱️ **Mouse**: Manage and toggle mouse sensitivity presets.
*   🌐 **Language**: English and Vietnamese.

2.  Run the application.

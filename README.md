## 🌟 Key Features

> [!TIP]
> **All capabilities listed below can be seamlessly combined, triggered, or orchestrated directly through the Macro Engine to fully automate your workflow.**

| Module | Key Capabilities | ⌨️ Macro Integration & Synergy |
| :--- | :--- | :--- |
| **⌨️ Macro Engine** | • Automate key sequences, mouse actions, loops, and conditions<br>• Supports logic variables and customizable delays | *The central orchestrator of all automated tasks.* |
| **👁️ Computer Vision** | • Screen image detection (OpenCV Template Matching)<br>• Pixel & color change monitoring in custom regions | *Trigger specific keystrokes or mouse clicks when targets or colors appear.* |
| **📝 OCR (Text Detection)** | • Real-time Optical Character Recognition directly from the screen<br>• Detects custom text strings, patterns, or numbers | *Fire automated macros instantly when a specific word or number is displayed.* |
| **🪟 Window Controller** | • Apply strict size and position presets for application windows<br>• Lock aspect ratios, zoom views, and target multiple windows<br>• **Live DWM Pinning**: Keep custom rounded/circular cropped window regions always on top | *Automatically refocus windows or adjust layout arrangements in real time.* |
| **🎙️ Audio Sensing** | • Monitor input levels from microphone or system audio channels<br>• Define custom amplitude/frequency activation thresholds | *Execute macros or system commands the moment a loud sound or trigger is detected.* |
| **🎵 Sound Effects** | • Trigger customizable notification sound presets<br>• Trim, manage, and catalog custom audio clips | *Play audio alerts to signal macro completion or status errors.* |
| **➕ Custom Crosshair** | • Render custom screen overlays (visual center crosshairs)<br>• Adjust shapes, size, colors, opacity, and visibility presets | *Toggle, show, or dynamically swap crosshair overlays based on macro state.* |
| **📐 Geometry Overlay** | • Draw lines, circles, boxes, and shapes directly on the screen overlay<br>• Visualize active click paths or target search zones | *Draw bounding boxes around detected targets while a macro is scanning.* |
| **🏷️ HUD Labels** | • Render dynamic text overlays, status indicators, or action logs on screen | *Display real-time visual progress or current execution steps of running macros.* |
| **📜 Script Command** | • Direct execution of CMD and PowerShell scripts | *Incorporate complex system-level scripts directly into macro actions.* |
| **🖱️ Mouse Presets** | • Configure and switch between custom cursor sensitivity and speed presets | *Automatically adjust DPI or cursor speeds during target tracking or click cycles.* |
| **🌐 Multilingual** | • Full interface support for English and Vietnamese | *Context-aware localization.* |

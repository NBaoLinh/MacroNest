## 🌟 Key Features

> [!TIP]
> **All capabilities listed below can be seamlessly combined, triggered, or orchestrated directly through the Macro Engine. Most modules support reading from or outputting to variables to enable dynamic automation logic.**

| Module | Key Capabilities & Variable Synergy | ⌨️ Macro Integration |
| :--- | :--- | :--- |
| **⌨️ Macro Engine** | • Automate key sequences, mouse actions, loops, and conditions<br>• Centralized variable management: read, write, and evaluate expressions | *The central orchestrator of all automation and logic.* |
| **👁️ Computer Vision** | • Screen image detection (OpenCV Template Matching)<br>• Pixel & color change monitoring in custom regions<br>• **Pixel Color Count**: Count pixels matching specific colors in a region<br>• *Variable Synergy: Saves found target coordinates (X, Y) and pixel counts to variables.* | *Trigger specific keystrokes or mouse clicks when visual targets or colors appear.* |
| **📝 OCR (Text Detection)** | • **Windows Native OCR**: Ultra-low latency Optical Character Recognition<br>• Detects custom text strings, patterns, or numbers on screen<br>• *Variable Synergy: Saves recognized text and coordinates to variables.* | *Fire automated macros instantly when specific words or numbers are detected.* |
| **🪟 Window Controller** | • Apply strict size and position presets for application windows<br>• Lock aspect ratios, zoom views, and target multiple windows<br>• **Live DWM Pinning**: Keep a cropped rectangular region of any window always on top<br>• *Variable Synergy: Reads target coordinates, dimensions, or window handles from variables.* | *Automatically refocus windows or adjust layout arrangements in real time.* |
| **🎙️ Audio Sensing** | • Monitor input levels from microphone or system audio channels<br>• Define custom amplitude/frequency activation thresholds<br>• *Variable Synergy: Saves audio level states and thresholds to variables.* | *Execute macros or system commands the moment a trigger volume is reached.* |
| **🎵 Sound Effects** | • Trigger customizable notification sound presets<br>• Trim, manage, and catalog custom audio clips | *Play audio alerts to signal macro completion or status errors.* |
| **➕ Custom Crosshair** | • Render built-in customizable center crosshairs (dot, cross, circle)<br>• Fine-tune size, colors, opacity, and toggle visibility presets | *Toggle or dynamically swap crosshair overlays based on macro state.* |
| **📐 Geometry Overlay** | • Draw lines, circles, boxes, and shapes directly on the screen overlay<br>• *Variable Synergy: Reads drawing coordinates dynamically from variables.* | *Draw bounding boxes around targets dynamically while macro is scanning.* |
| **🏷️ HUD Labels** | • Render custom text labels, active timers, and countdowns on screen overlay<br>• *Variable Synergy: Dynamically displays the current values of variables in real time.* | *Display visual progress, elapsed time, or current execution steps of macros.* |
| **📜 Script Command** | • Direct execution of CMD and PowerShell scripts | *Incorporate complex system-level scripts directly into macro actions.* |
| **🖱️ Mouse & Hardware** | • **Interception & Arduino**: Driver bypass and hardware-level macros for anti-cheat avoidance<br>• **Mouse Path**: Record, replay, and manage custom cursor movement trajectories<br>• Configure cursor sensitivity and speed presets<br>• *Variable Synergy: Reads target coordinates from variables for auto-aim/clicks.* | *Automatically adjust DPI or replay mouse paths during automation cycles.* |
| **🌐 Supported Languages** | • Full interface support for English and Vietnamese | *Context-aware localization.* |
